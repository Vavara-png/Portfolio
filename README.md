# 🌐 Portfólio de Fernando

Este é meu portfólio profissional desenvolvido em **React**, com animações modernas e design responsivo.  
Aqui você encontrará meus principais projetos, habilidades e formas de contato.

---

## 🚀 Tecnologias Utilizadas
- React
- Styled Components
- Framer Motion
- React Router
- React Icons

---

## 📂 Estrutura do Projeto
- **Home:** resumo profissional
- **Sobre Mim:** apresentação, foto e contatos
- **Projetos:** lista de projetos com descrição, tecnologias e links
- **Habilidades:** principais tecnologias adquiridas
- **Contato:** formulário funcional + redes sociais
